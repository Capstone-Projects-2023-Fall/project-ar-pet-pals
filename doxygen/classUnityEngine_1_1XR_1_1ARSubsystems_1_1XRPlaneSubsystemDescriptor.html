Detailed Description

Indicates the capabilities supported by a provider of the XRPlaneSubsystem. Provider implementations must derive from XRPlaneSubsystem.Provider and may override virtual class members.

bool UnityEngine.XR.ARSubsystems.XRPlaneSubsystemDescriptor.supportsArbitraryPlaneDetection
get

Indicates whether the provider implementation supports the detection of planes that are aligned with neither the horizontal nor vertical axes. If false, BoundedPlane trackables returned by XRPlaneSubsystem.GetChanges must not have their BoundedPlane.alignment value set to PlaneAlignment.NotAxisAligned.

true if the implementation supports the detection of planes oriented at arbitrary angles. Otherwise, false.
