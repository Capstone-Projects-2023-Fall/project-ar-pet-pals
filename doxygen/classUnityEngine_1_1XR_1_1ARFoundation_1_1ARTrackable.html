Detailed Description

The base class for all ARTrackable<TSessionRelativeData,TTrackable> types.

A generic component for trackables. A "trackable" is a feature in the physical environment that can be detected and tracked by an XR device.

A "trackable" is something that is tracked in the physical environment. These include:

  • ARAnchor
  • AREnvironmentProbe
  • ARFace
  • ARHumanBody
  • ARParticipant
  • ARPlane
  • ARPointCloud
  • ARRaycast
  • ARTrackedImage
  • ARTrackedObject

bool UnityEngine.XR.ARFoundation.ARTrackable< TSessionRelativeData, TTrackable >.destroyOnRemoval
getset

If true, this component's GameObject will be removed immediately when the XR device reports this trackable is no longer tracked.

Setting this to false will keep the GameObject around. You might want to do this, for example, if you have custom removal logic, such as a fade out.
