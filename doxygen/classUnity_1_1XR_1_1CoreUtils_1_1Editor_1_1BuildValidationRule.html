Detailed Description

Defines a validation rule used for assessing package setup correctness.

Use CheckPredicate to define the validation logic. Use FixIt to provide logic correct the validation problem. The other properties of BuildValidationRule define when the rule function is executed and how the rule is displayed in the Editor.

See [Project Validation](xref:xr-core-utils-project-validation) for more information.

bool Unity.XR.CoreUtils.Editor.BuildValidationRule.SceneOnlyValidation
getset

Whether to prevent this build rule from running when the Editor is in the [Prefab mode](xref:EditingInPrefabMode)).

Set this property to true when you have a rule that checks whether a Scene is setup correctly, but which fails in the special Editor Prefab mode. Set this property to false for other, general purpose rules.
