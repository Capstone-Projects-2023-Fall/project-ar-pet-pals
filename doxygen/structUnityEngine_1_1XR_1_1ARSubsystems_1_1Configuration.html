Detailed Description

Represents a session configuration. It consists of a configuration descriptor, which contains information about the capabilities of the configuration, and the specific features which should be enabled by this configuration. Use XRSessionSubsystem.DetermineConfiguration(Feature) to get a Configuration given a set of features.

See also
ConfigurationChooser, XRSessionSubsystem.configurationChooser, XRSessionSubsystem.DetermineConfiguration(Feature)

descriptorA ConfigurationDescriptor for this configuration.
featuresA set of Feature(s) that should be enabled for this configuration. You can only enable exactly zero or one camera modes (see Feature.UserFacingCamera and Feature.WorldFacingCamera). If zero camera modes are enabled, no camera texture will be available. Some platforms might support a configuration that does not provide camera textures, which can be more performant if they are not necessary. All features must be supported by the descriptor .
Exceptions
System.InvalidOperationExceptionThrown if multiple camera modes are enabled.
System.InvalidOperationExceptionThrown if multiple tracking modes are enabled.
System.NotSupportedExceptionThrown if the descriptor does not support one or more features .
