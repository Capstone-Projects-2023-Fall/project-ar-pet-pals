Method to be implemented by the provider to get the human body pose 2D joints for the current frame.

Parameters
defaultHumanBodyPose2DJointThe default value for the body pose 2D joint.
screenWidthThe width of the screen, in pixels.
screenHeightThe height of the screen, in pixels.
screenOrientationThe orientation of the device so that the joint positions can be adjusted as required.
allocatorThe allocator to use for the returned array memory.
Returns
The array of body pose 2D joints.

The returned array can be empty if the system is not enabled for human body pose 2D or if the system does not detect a human in the camera image.

Exceptions
System.NotSupportedExceptionThrown if the implementation does not support human body pose 2D.
