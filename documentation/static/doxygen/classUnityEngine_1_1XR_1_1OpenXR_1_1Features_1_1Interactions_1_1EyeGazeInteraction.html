This OpenXRInteractionFeature enables the use of eye gaze interaction profiles in OpenXR. It enables XR_EXT_eye_gaze_interactionin the underlying runtime. This creates a new InputDevice with the InputDeviceCharacteristics.EyeTracking characteristic. This new device has both EyeTrackingUsages.gazePosition and EyeTrackingUsages.gazeRotation input features, as well as CommonUsages.isTracked and CommonUsages.trackingState usages to determine if the gaze is available. Detailed Description

This OpenXRInteractionFeature enables the use of eye gaze interaction profiles in OpenXR. It enables XR_EXT_eye_gaze_interaction

in the underlying runtime. This creates a new InputDevice with the InputDeviceCharacteristics.EyeTracking characteristic. This new device has both EyeTrackingUsages.gazePosition and EyeTrackingUsages.gazeRotation input features, as well as CommonUsages.isTracked and CommonUsages.trackingState usages to determine if the gaze is available.
