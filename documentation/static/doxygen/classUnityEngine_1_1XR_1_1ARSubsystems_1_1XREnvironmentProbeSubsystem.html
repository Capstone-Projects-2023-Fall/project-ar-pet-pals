◆ RemoveEnvironmentProbe()

bool UnityEngine.XR.ARSubsystems.XREnvironmentProbeSubsystem.RemoveEnvironmentProbe ( TrackableId  trackableId)

Asynchronously removes the environment probe matching the trackable ID from the AR session.

Parameters
trackableIdTrackable ID of the environment probe to be removed from the AR session.
Returns
true if the environment probe is found in the current AR session and will be removed. Otherwise, false.

RemoveEnvironmentProbe can be used to remove both manually placed and automatically placed environment probes if the implementation supports such removals, as indicated by the descriptor properties XREnvironmentProbeSubsystemDescriptor.supportsRemovalOfManual and XREnvironmentProbeSubsystemDescriptor.supportsRemovalOfAutomatic.

Exceptions
System.NotSupportedExceptionThrown for platforms that do not support removal of the type of environment probe.

bool UnityEngine.XR.ARSubsystems.XREnvironmentProbeSubsystem.automaticPlacementRequested
getset

Specifies whether the AR session should automatically place environment probes in the scene.

true if automatic placement of environment probes is enabled. Otherwise, false.

If both manual and automatic placement of environment probes are supported, manually placed environment probes can be specified via TryAddEnvironmentProbe regardless of whether automatic placement is enabled or not.

Exceptions
System.NotSupportedExceptionThrown when setting this value to true for implementations that do not support automatic placement.
