Detailed Description

Stores a package version in a logically comparable format.

The version information stored in the PackageVersion follows the Semantic Versioning Specification (SemVer) https://semver.org/. The version consists of a MajorVersion.MinorVersion.PatchVersion numerical value followed by optional -Prerelease+BuildMetaData version information.

PackageVersion follows all the standard for valid and invalid formatting of a version except for limiting the MajorVersion, MinorVersion, and PatchVersion to the ulong.MaxValue. Their is no such restriction for the values of Prerelease or BuildMetaData.
