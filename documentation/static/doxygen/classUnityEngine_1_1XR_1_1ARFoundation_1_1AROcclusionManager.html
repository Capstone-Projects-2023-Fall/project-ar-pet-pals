Attempt to get the latest raw environment depth CPU image. This provides direct access to the raw pixel data.

‍[!NOTE] The XRCpuImage must be disposed to avoid resource leaks.

This differs from TryAcquireEnvironmentDepthCpuImage in that it always tries to acquire the raw environment depth image, whereas TryAcquireEnvironmentDepthCpuImage depends on the value of environmentDepthTemporalSmoothingEnabled.

Parameters
cpuImageIf this method returns true, an acquired XRCpuImage.
Returns
Returns true if the CPU image was acquired. Returns false otherwise.
