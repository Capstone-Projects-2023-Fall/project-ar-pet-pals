Detailed Description

Write accessor to a composite binding.

TODO: remove this and merge it into BindingSyntax

To add a composite binding to an action, you must first call InputActionSetupExtensions.AddCompositeBinding and then use the CompositeSyntax struct to add composite parts.

playerInput.actions["fire"]
.ChangeBinding(0)
.AddCompositeBinding("2DVector")
.With("Up", "<Keyboard>/w")
.With("Down", "<Keyboard>/s")
.With("Left", "<Keyboard>/a")
.With("Right", "<Keyboard>/d");
See also
AddBinding(InputAction,InputBinding), ChangeBinding(InputAction,int)

int UnityEngine.InputSystem.InputActionSetupExtensions.CompositeSyntax.bindingIndex

Index of the binding that the accessor refers to.

When accessing bindings on an InputAction, this is the index in InputAction.bindings of the action. When accessing bindings on an InputActionMap, it is the index InputActionMap.bindings of the map.
