Gets the boundary vertices of a given BoundedPlane based on its BoundedPlane.trackableId.

Parameters
trackableIdThe TrackableId of a BoundedPlane.
allocatorAn Allocator to use if boundary needs to be created. Allocator.Temp is not supported. Use Allocator.TempJob if you need temporary memory.
boundaryThe boundary vertices will be stored here. If the given array is the same length as the number of vertices, it is overwritten with the new data. Otherwise, it is disposed and recreated with the correct length.

See CreateOrResizeNativeArrayIfNecessary<T> for a helper method implementation.
