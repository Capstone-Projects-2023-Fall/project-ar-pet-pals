◆ StartPlayback()

ArStatus UnityEngine.XR.ARCore.ARCoreSessionSubsystem.StartPlayback ( string  playbackDataset)

Starts playing back a previously recorded session. (see StartRecording)

The begin playback, the session must first be paused, the playback dataset set, then resumed. This method does all of those things for you, but can take significant time (.5 - 1 seconds) to do so.

Parameters
playbackDatasetThe path to an mp4 file previously recorded using StartRecording.
Returns
Returns ArStatus.Success if successful. Returns one of the following otherwise:
  • Returns ArStatus.ErrorSessionUnsupported if playback is incompatible with selected features.
  • Returns ArStatus.ErrorPlaybackFailed if an error occurred with the MP4 dataset file such as not being able to open the file or the file is unable to be decoded.
See also
StartRecording, StopRecording, StopPlayback
