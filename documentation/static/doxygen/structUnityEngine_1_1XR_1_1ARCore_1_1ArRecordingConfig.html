Detailed Description

A recording configuration struct that contains the configuration for session recording. See ArSession.StartRecording(ArRecordingConfig).

A ArRecordingConfig represents a native object that must be disposed (by calling ArRecordingConfig.Dispose) to prevent memory leaks. Consider using a using statement for convenience:

This is a C# wrapper for ARCore's native ArRecordingConfig
