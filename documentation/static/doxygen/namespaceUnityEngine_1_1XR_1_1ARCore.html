◆ ArCameraConfigDepthSensorUsage

enum UnityEngine.XR.ARCore.ArCameraConfigDepthSensorUsage

Depth sensor usage filters for ArCameraConfigFilter.

Enumerator
RequireAndUse 

Filters for camera configs that require a depth sensor to be present on the device, and that will be used by ARCore. See ArCameraConfigFilter

See ARCore supported devices for a list of devices that currently have supported depth sensors.

DoNotUse 

Filters for camera configs where a depth sensor is not present, or is present but will not be used by ARCore.

Used by ArCameraConfigFilter.SetDepthSensorUsage and ArCameraConfigFilter.GetDepthSensorUsage to filter camera configurations when the app requires exclusive access to the depth sensor outside of ARCore, for example to support 3D mesh reconstruction. Available on all ARCore supported devices.

◆ ArCameraConfigTargetFps

enum UnityEngine.XR.ARCore.ArCameraConfigTargetFps

Target camera capture frame rates.

The target frame rate represents the maximum or desired frame rate. Actual camera capture frame rates can be lower than the target frame rate under low light conditions in order to accommodate longer exposure times.

See also
ArCameraConfigFilter
Enumerator
Fps30 

Target 30fps camera capture frame rate.

Available on all ARCore supported devices.

Used by ArCameraConfigFilter.SetTargetFps and ArCameraConfigFilter.GetTargetFps.

Fps60 

Target 60fps camera capture frame rate.

Increases power consumption and may increase app memory usage.

See the ARCore supported devices page for a list of devices that currently support 60fps.

Used by ArCameraConfigFilter.SetTargetFps and ArCameraConfigFilter.GetTargetFps.
