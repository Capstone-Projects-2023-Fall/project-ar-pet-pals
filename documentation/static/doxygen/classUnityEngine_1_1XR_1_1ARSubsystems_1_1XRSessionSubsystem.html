Detailed Description

This subsystem controls the lifecycle of an XR session. Some platforms, particularly those that have non-XR modes, need to be able to turn the session on and off to enter and exit XR modes of operation.

◆ DetermineConfiguration()

Configuration? UnityEngine.XR.ARSubsystems.XRSessionSubsystem.DetermineConfiguration ( Feature  features)
inline

Determines the Configuration the session will use given the requested features .

This method uses the current configurationChooser to choose a configuration using the requested features and the capabilities of the available configurations. The configuration chooser is customizable, see ConfigurationChooser. If you do not set a configuration chooser, the DefaultConfigurationChooser is used.

You can use this method to determine what would happen if you were to enable or disable a particular feature. For example, you can use this method to determine which features would be enabled if you were to change the camera mode from Feature.WorldFacingCamera to Feature.UserFacingCamera.

Parameters

Returns
The Configuration the session would use given the requested features , or null if configuration introspection is not available.

◆ GetConfigurationDescriptors()

NativeArray< ConfigurationDescriptor > UnityEngine.XR.ARSubsystems.XRSessionSubsystem.GetConfigurationDescriptors ( Allocator  allocator)

Get the list of supported configuration descriptors. The session can have multiple, discrete modes of operation. A configuration represents the capabilities of a mode of operation, which can be a subset of the session's overal capabilities. That is, the session might support many features, but not all at the same time. This is used by XRSessionSubsystem.DetermineConfiguration(Feature) to determine the best configuration given a set of requested features.

Parameters
allocatorThe allocator to use for the returned NativeArray.
Returns
Allocates a new NativeArray and populates it with descriptors describing the supported configurations. The caller owns the memory and is responsible for calling Dispose on the NativeArray.
