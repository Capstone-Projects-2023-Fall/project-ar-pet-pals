NativeArray< ConfigurationDescriptor > GetConfigurationDescriptors (Allocator allocator)
 Get the list of supported configuration descriptors. The session can have multiple, discrete modes of operation. A configuration represents the capabilities of a mode of operation, which can be a subset of the session's overal capabilities. That is, the session might support many features, but not all at the same time. This is used by XRSessionSubsystem.DetermineConfiguration(Feature) to determine the best configuration given a set of requested features.
