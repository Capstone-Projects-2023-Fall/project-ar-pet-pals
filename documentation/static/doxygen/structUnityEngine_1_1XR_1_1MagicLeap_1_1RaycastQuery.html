Constructs a raycast query.

If width or height are greater than 1, width * height rays are cast from the ray origin within a frustum defined by horizontalFov and up .

Parameters
rayThe ray, in session space, to cast.
upIf multiple rays are to be fired (i.e., width or height are greater than 1), this is used to determine the coordinate system used to calculate the directions of those rays; therefore must be orthogonal to the direction vector. This parameter is has no effect on a single-point raycast.
widthThe number of horizontal rays to cast. Must be at least 1.
heightThe number of vertical rays to cast. Must be at least 1.
horizontalFovThe horizontal field of view, in degrees. When width or height are greater than 1, this is used to determine the frustum in which to cast multiple rays.
collideWithUnobservedIf true, a ray will temrinate when encountering an unobserved area and return a surface or the ray will continue until it ends or hits an observed surface.
Exceptions
System.ArgumentOutOfRangeExceptionThrown if width is less than 1.
System.ArgumentOutOfRangeExceptionThrown if height is less than 1.
System.ArgumentExceptionThrown if width or height is greater than 1 but up is not orthogonal to the ray 's direction.
System.ArgumentExceptionThrown if width or height is greater than 1, but up is zero.
