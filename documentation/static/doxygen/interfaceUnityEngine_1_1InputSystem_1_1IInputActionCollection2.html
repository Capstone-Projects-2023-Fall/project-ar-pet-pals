)

Find an InputAction in the collection by its InputAction.name or by its InputAction.id (in string form).

Parameters
actionNameOrIdName of the action as either a "map/action" combination (e.g. "gameplay/fire") or a simple name. In the former case, the name is split at the '/' slash and the first part is used to find a map with that name and the second part is used to find an action with that name inside the map. In the latter case, all maps are searched in order and the first action that has the given name in any of the maps is returned. Note that name comparisons are case-insensitive.

Alternatively, the given string can be a GUID as given by InputAction.id.

Parameters
throwIfNotFoundIf true, instead of returning null when the action cannot be found, throw ArgumentException.
Returns
The action with the corresponding name or null if no matching action could be found.
Exceptions
ArgumentNullExceptionactionNameOrId is null.
ArgumentExceptionThrown if throwIfNotFound is true and the action could not be found. -Or- If actionNameOrId contains a slash but is missing either the action or the map name.

Find the index of the first binding that matches the given mask.

Parameters
maskA binding. See InputBinding.Matches for details.
actionReceives the action on which the binding was found. If none was found, will be set to null.
Returns
Index into InputAction.bindings of action of the binding that matches mask . If no binding matches, will return -1.

For details about matching bindings by a mask, see InputBinding.Matches.

new InputBinding { path = "<Gamepad>/buttonSouth" },
out var action);
