Constructs a TrackableChanges<T> from native memory.

Because native code might be using an older version of T , this constructor first fills the added and updated arrays with copies of defaultT before copying the data from the addedPtr and updatedPtr pointers. This ensures that the addition of new fields to T will have appropriate default values.

Parameters
addedPtrA pointer to a block of memory containing addedCount elements each of size stride .
addedCountThe number of added elements.
updatedPtrA pointer to a block of memory containing updatedCount elements each of size stride .
updatedCountThe number of updated elements.
removedPtrA pointer to a block of memory containing removedCount TrackableIds.
removedCountThe number of removed elements.
defaultTA default T which should be used to fill the added and updated arrays before copying data from addedPtr and updatedPtr , respectively.
strideThe number of bytes for each element in the addedPtr and updatedPtr arrays.
allocatorAn allocator to use when creating the added, updated, and removed arrays.
