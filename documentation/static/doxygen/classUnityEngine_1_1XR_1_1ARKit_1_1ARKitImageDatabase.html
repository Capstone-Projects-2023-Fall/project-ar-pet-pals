This method should schedule a Unity Job which adds an image to this reference image library.

Parameters
imageBytesThe raw image bytes in format . You can assume the bytes are valid until the job completes.
sizeInPixelsThe width and height of the image, in pixels.
formatThe format of imageBytes . The format has already been validated with IsTextureFormatSupported(TextureFormat).
referenceImageThe XRReferenceImage data associated with the image to add to the library. This includes information like physical dimensions, associated Texture2D (optional), and string name.
inputDepsA JobHandle that represents input dependencies for the add image job.
Returns
Returns an AddReferenceImageJobState which contains the state of the asynchronous image addition.
Exceptions
System.NotImplementedExceptionThrown by this base class implementation. If supportsValidation is true, then this method should be implemented by the derived class.
